## Quick Diagnostics

```bash
# Container status
docker compose ps

# View logs (last 50 lines)
docker compose logs --tail=50 graylog

# Follow logs
docker compose logs -f graylog

# Exec into container
docker compose exec graylog bash
```

## Common Issues

### Container fails to start

1. Check logs: docker compose logs graylog
2. Verify environment variables are set correctly
3. Check database connectivity: pg_isready -h lab-db1 -p 5432
4. Verify port is not already in use: ss -tlnp | grep 9000

### Authentication fails (SSO)

1. Verify Keycloak client is configured: https://lab-id1:8443/admin/
2. Check client secret matches environment variable
3. Verify redirect URIs match exactly
4. Check Keycloak realm is it-stack

### Database connection error

```bash
# Test connectivity
psql -h lab-db1 -U graylog_user -d graylog_db -c '\conninfo'

# Check pg_hba.conf allows connection from lab-proxy1
```

### Performance issues

1. Check resource usage: docker stats it-stack-graylog
2. Verify Redis is reachable: edis-cli -h lab-db1 ping
3. Check Elasticsearch if used: curl http://lab-db1:9200/_cluster/health
